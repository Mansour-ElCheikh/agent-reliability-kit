# epic.md

**Status:** Approved

## Problem
Support staff cannot get structured logs out of the app for a bug report.

## Goal
Export the in-memory log buffer as a JSON array, newest first, honoring the active level filter.

## Components
| # | Component | What |
|---|---|---|
| 001 | Formatter | Shape each entry into a JSON object |
| 002 | Exporter | Assemble, filter, and order the array |

## Acceptance Criteria
- Each exported entry has a timestamp, a level, and a message.
- The output parses as a JSON array.
- Entries below the active level filter are omitted.

# epic.md

**Status:** Approved

## Problem
Users import contacts from CSV files exported by other tools. There is no importer, so they re-key data by hand.

## Goal
A CSV import command that maps columns to contact fields and reports rejected rows. Done when a 1,000-row file imports in under 2 seconds with a per-row error report.

## Components
| # | Component | What |
|---|---|---|
| 001 | Parser | Read the CSV text into rows |
| 002 | Mapper | Map columns to contact fields |

## Acceptance Criteria
- A well-formed 1,000-row file imports with zero errors.
- A row missing a required field is rejected and listed in the error report.
- The importer returns the count of imported and rejected rows.

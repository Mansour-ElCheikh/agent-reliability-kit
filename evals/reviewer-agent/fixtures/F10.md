# epic.md

**Status:** Approved

## Problem
The search box queries the server on every keystroke, hammering the API.

## Goal
Debounce the search input so a query fires only after the user pauses typing.

## Components
| # | Component | What |
|---|---|---|
| 001 | Debouncer | Delay and coalesce rapid calls |
| 002 | SearchBox | Wire the input to the debounced query |

## Acceptance Criteria
- A query fires 300 ms after the last keystroke.
- Typing five characters quickly fires exactly one query.
- Clearing the input cancels any pending query.

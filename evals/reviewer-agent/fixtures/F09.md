# epic.md

**Status:** Approved

## Problem
Users want to share a filtered table view but can only screenshot it.

## Goal
Copy the current filtered results to the clipboard as tab-separated text.

## Components
| # | Component | What |
|---|---|---|
| 001 | Serializer | Turn the filtered rows into TSV |
| 002 | Clipboard | Write the text to the clipboard |

## Acceptance Criteria
- The copied text has one line per visible (filtered) row.
- Columns are tab-separated in the current column order.
- Copying an empty result set writes an empty string, not an error.

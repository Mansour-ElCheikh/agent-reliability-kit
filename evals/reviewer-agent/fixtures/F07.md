**Status:** Approved

## Problem
Bulk uploads accept malformed records, which corrupt downstream reports.

## Goal
Validate each record on upload and reject the batch if any record is invalid, with a per-record reason.

## Components
| # | Component | What |
|---|---|---|
| 001 | Reader | Read the upload into records |
| 002 | Validator | Validate each record and report failures |

## Acceptance Criteria
- A batch of valid records is accepted in full.
- A record missing a required field is reported with the failing field name.
- A batch with any invalid record is rejected as a whole.

# epic.md

**Status:** Approved

## Problem
Sessions never expire, so a stolen token works forever.

## Goal
Expiring sessions with sliding refresh. Done when an idle session expires after 30 minutes and an active one refreshes transparently.

## Components
| # | Component | What |
|---|---|---|
| 001 | Store | Persist sessions with expiry |
| 002 | Guard | Check and refresh on each request |

## Acceptance Criteria
- A session older than 30 idle minutes is treated as expired.
- An active request within the window extends the expiry.
- An expired session forces re-authentication.

## Problem
There is no way to export the ledger for an accountant; users screenshot the table.

## Goal
Export the ledger to CSV with one row per transaction and a header row.

## Components
| # | Component | What |
|---|---|---|
| 001 | Exporter | Serialize transactions to CSV |

## Acceptance Criteria
- The export writes one row per transaction.
- The first line is a header row naming each column.
- Amounts are written with two decimal places.

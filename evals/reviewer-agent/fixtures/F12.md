# epic.md

**Status:** Approved

## Problem
Users pick weak passwords and have no feedback until login fails later.

## Goal
A password strength meter that scores a candidate and explains the weakest factor.

## Components
| # | Component | What |
|---|---|---|
| 001 | Scorer | Compute a 0-4 strength score |
| 002 | Advice | Name the weakest factor |

## Acceptance Criteria
- A password under 8 characters scores 0 (too short).
- A known common password scores 0 regardless of length.
- A 16-character mixed-class password scores 4.
